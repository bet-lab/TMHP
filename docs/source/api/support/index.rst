===============
Support modules
===============

Lower-level building blocks the system models compose internally. You
rarely need to import these directly, but knowing what's available
makes it easier to extend a model or build a custom one. Each
subsection groups modules that share a single concern — refrigerant
state and cycle analysis, heat-transfer components, demand and
weather inputs, and so on.
