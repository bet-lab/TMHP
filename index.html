Physics-Based Heat Pump Models¶

First-principles dynamic models for air-source, ground-source, and water-source heat pump systems.

physics_hp solves the thermodynamic refrigerant cycle at every time step using CoolProp, so the same model code applies across refrigerants and operating envelopes without needing manufacturer-specific curve fits.
